# declarative.space hackathon @ PlanetNix 2026

Nix permits self-hosting. Now make it for everyone.

**When:** March 5--6, 2026
**Where:** [PlanetNix](https://planetnix.com) @ Pasadena Convention Center, CA
**Website:** <https://declarative.space>

## How it works

1. Show up at PlanetNix on March 5. Form a team (1--4 people) or go solo.
2. Build something that advances self-hosting.
3. Demo your project live on March 6 at 4 pm.
4. [Register your team](https://github.com/numtide/planetnix-hackathon/issues/new?template=submission.yml) via GitHub Issues. Teams are called by name for demos.

## Judging criteria

| Criterion   | Weight |
|-------------|--------|
| Self-hosting | 30%    |
| Usability   | 30%    |
| Creativity  | 20%    |
| Execution   | 20%    |
